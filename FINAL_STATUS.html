🚀 הוראות הרצה סופיות:

  1. התקן Node.js:
    • הורד מ-nodejs.org
    • או הרץ: winget install OpenJS.NodeJS
  2. אימות התקנה:
    node --version
    npm --version
  3. הרצת האפליקציה:
    cd campus-web-react
    npm install
    npm start
  4. גישה לאפליקציה:

    האפליקציה תיפתח אוטומטית ב: http://localhost:3000
