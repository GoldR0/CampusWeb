ONO - מערכת ניהול קמפוס

מדריך התקנה והרצה

✅ תיקונים שבוצעו בהצלחה:

  • תיקון מבנה JSX - הוספת תגי סגירה חסרים
  • הוספת סקציית פרופיל אישי מלאה
  • תיקון שגיאות TypeScript בהשוואות מחרוזות
  • תיקון תנאים לזיהוי מסכי ניהול
  • וידוא שכל התגים נסגרים כראוי
  • הקוד מוכן להרצה!

⚠️ בעיה זוהתה:

Node.js לא מותקן במערכת שלך.

כדי להריץ את האפליקציה, יש להתקין Node.js תחילה.

שלב 1: התקנת Node.js

Node.js הוא סביבת הרצה ל-JavaScript שנדרשת להרצת פרויקטי React.

אפשרות א' - הורדה ישירה:
הורד Node.js מהאתר הרשמי
אפשרות ב' - התקנה דרך Chocolatey (אם מותקן):
choco install nodejs
אפשרות ג' - התקנה דרך Winget:
winget install OpenJS.NodeJS

שלב 2: אימות ההתקנה

לאחר ההתקנה, פתח טרמינל חדש ובדוק:

node --version npm --version

אם הפקודות עובדות, Node.js מותקן כראוי.

שלב 3: הרצת הפרויקט

נווט לתיקיית הפרויקט והרץ את הפקודות הבאות:

cd campus-web-react npm install npm start

האפליקציה תיפתח אוטומטית בדפדפן בכתובת: http://localhost:3000

ℹ️ מידע על הפרויקט:

  • טכנולוגיות: React 18, TypeScript, Material-UI
  • תמיכה: עברית עם כיוון RTL
  • גודל: מעל 4,000 שורות קוד
  • תכונות: מערכת ניהול קמפוס מקיפה

🎉 תכונות האפליקציה:

🏠 מסך בית

ניווט מהיר לכל השירותים, סטטיסטיקות, הודעות חשובות

👤 פרופיל אישי

פרטים אישיים, סטטיסטיקות, היסטוריית פעילות

📚 מרכז לימודים

ניהול מטלות, לוח זמנים, חומרי לימוד

🔍 מציאות ואבדות

דיווח פריטים אבודים, חיפוש פריטים שנמצאו

🛒 שוק יד שנייה

מכירה וקנייה של פריטים, ניהול מודעות

🛠️ בקשות שירות

דיווח בעיות, מעקב אחר בקשות, תמיכה טכנית

💬 פורום קורס

דיונים, שאלות ותשובות, שיתוף מידע

☕ קפיטריה

הזמנת אוכל, תפריטים, מעקב הזמנות

🎉 אירועים קהילתיים

ארגון אירועים, הרשמה, ניהול משתתפים

❓ עזרה ותמיכה

כרטיסי עזרה, מדריכים, תמיכה טכנית

⚙️ מסכי ניהול

ניהול מלא לכל המערכות, סטטיסטיקות, בקרה

🔧 פקודות שימושיות:

  • npm start - הרצת האפליקציה במצב פיתוח
  • npm run build - בניית האפליקציה לייצור
  • npm test - הרצת בדיקות
  • npm run eject - יצירת קונפיגורציה מותאמת (לא מומלץ)

🚀 מוכן להרצה!

לאחר התקנת Node.js והרצת הפקודות, האפליקציה תהיה זמינה עם כל התכונות המתוארות למעלה.

זכור: יש להריץ npm install פעם אחת בלבד לפני npm start.
