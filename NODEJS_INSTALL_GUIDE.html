📥 מדריך התקנת Node.js

התקנה ידנית ב-Windows

⚠️ בעיה זוהתה:

Node.js לא מותקן במערכת שלך.

זה הסיבה ש-npm לא עובד. בואו נתקין אותו עכשיו!

שלב 1: הורדת Node.js

  1. לך לאתר הרשמי של Node.js: nodejs.org
  2. בדף הראשי תראה שתי גרסאות:
    • LTS (מומלץ) - גרסה יציבה וארוכת טווח
    • Current - הגרסה העדכנית ביותר
  3. לחץ על הכפתור הירוק של LTS (מומלץ)
  4. הקובץ יורד אוטומטית (בערך 20MB)

שלב 2: התקנת Node.js

  1. מצא את הקובץ שהורדת (בדרך כלל בתיקיית Downloads)
  2. הקובץ ייקרא משהו כמו: node-v22.18.0-x64.msi
  3. לחץ פעמיים על הקובץ כדי להתחיל את ההתקנה
  4. אם מופיעה הודעת אבטחה, לחץ "Yes" או "הפעל"
  5. בחלון ההתקנה:
    • לחץ "Next" בכל השלבים
    • וודא שהאפשרות "Add to PATH" מסומנת
    • השאר את כל ההגדרות ברירת המחדל
  6. לחץ "Install" והמתן לסיום ההתקנה
  7. לחץ "Finish"

שלב 3: אימות ההתקנה

לאחר ההתקנה, סגור את הטרמינל הנוכחי ופתח טרמינל חדש.

הרץ את הפקודות הבאות:

node --version
npm --version

אם הכל תקין, תראה משהו כמו:

v22.18.0
10.2.4

שלב 4: הרצת האפליקציה

עכשיו שאתה יכול להשתמש ב-npm, הרץ את הפקודות הבאות:

cd campus-web-react
npm install
npm start

האפליקציה תיפתח אוטומטית בדפדפן ב: http://localhost:3000

💡 טיפים חשובים:

  • וודא שהאפשרות "Add to PATH" מסומנת בזמן ההתקנה
  • סגור ופתח טרמינל חדש לאחר ההתקנה
  • אם עדיין לא עובד, נסה הפעלה מחדש של המחשב
  • אם יש בעיות, בדוק שהורדת את הגרסה הנכונה (64-bit)

🎯 מה קורה אחרי ההתקנה:

  • תוכל להריץ node ו-npm מכל מקום
  • תוכל להתקין חבילות עם npm install
  • תוכל להריץ פרויקטי React עם npm start
  • האפליקציה שלך תהיה זמינה ב-localhost:3000

🔧 אם עדיין יש בעיות:

  1. הפעל מחדש את המחשב - לפעמים זה נדרש
  2. בדוק את משתנה הסביבה PATH:
    • לחץ Win+R, הקלד sysdm.cpl
    • לך לטאב "Advanced"
    • לחץ "Environment Variables"
    • בדוק שב-PATH יש: C:\Program Files\nodejs\
  3. נסה להתקין גרסה אחרת של Node.js
  4. בדוק שהורדת את הגרסה הנכונה (64-bit ולא 32-bit)
